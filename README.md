# 🎮 Game Booster

Optimizador de PC para gaming desarrollado en Python con interfaz gráfica.

## ¿Qué hace?
- Limpia archivos temporales del sistema (caché)
- Cierra procesos en segundo plano (Chrome, Discord)
- Libera RAM para mejorar el rendimiento

## Tecnologías
- Python 3
- Tkinter (interfaz gráfica)
- PyInstaller (compilado a .exe)

## Cómo usarlo
1. Ejecutar `main.py` con Python o el `.exe` compilado
2. Presionar el botón **OPTIMIZAR**
3. Esperar el mensaje de confirmación
4. Abrir el juego

## ⚠️ Importante
Guardar los trabajos antes de optimizar, ya que se cerrarán programas abiertos.
