GAME BOOSTER - GUÍA DE USO

Este programa optimiza el rendimiento del sistema antes de jugar.

FUNCIONES:
- Limpia archivos temporales (cache del sistema)
- Cierra procesos en segundo plano (Chrome, Discord, etc.)
- Libera memoria RAM

CÓMO USAR:
1. Abrir el programa
2. Presionar el botón "OPTIMIZAR"
3. Esperar mensaje de confirmación
4. Ejecutar el juego

RECOMENDACIONES:
- Usar antes de iniciar un juego
- Guardar trabajos antes de optimizar (se cerrarán programas)

IMPORTANTE:
Este programa no aumenta FPS directamente, pero mejora el rendimiento general del sistema.
